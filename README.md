# Seelam Manjunath Reddy - Portfolio Website

Live Website: https://manjunathreddy078.github.io/

## Overview
This repository contains a static personal portfolio website focused on:
- Academic projects
- Certifications gallery
- Internship readiness profile
- Poster and presentation contributions

The website is built using plain HTML, CSS, and JavaScript for easy maintenance and fast GitHub Pages deployment.

## Project Structure
- `index.html` - Home page, profile, summary, education, skills, achievements, and contact
- `projects.html` - Academic projects and design contributions
- `certificates.html` - Top certificates + expandable full certificate list
- `style.css` - Global theme, layout, responsive styles, UI effects
- `script.js` - Certificates section interaction
- `assets/images/profile.jpg` - Profile image
- `assets/docs/resume_latest.pdf` - Resume file for website CTA

## Run Locally
From repository root:

```powershell
python -m http.server 5500
```

Open in browser:
- `http://localhost:5500/index.html`

## Update Content
- Update personal details in `index.html`
- Update projects in `projects.html`
- Update certifications in `certificates.html`
- Replace resume file at `assets/docs/resume_latest.pdf` (keep filename same)

## Deployment
This site is deployed via GitHub Pages from the `main` branch (root).

Typical publish flow:

```powershell
git add .
git commit -m "Update portfolio content"
git push origin main
```
